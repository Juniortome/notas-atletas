Projeto: cálculo de médias válidas de atletas
# notas-atletas

Projeto em JavaScript que recebe o nome dos atletas e as notas de 5 jurados e calcula a média válida,
desconsiderando a maior e a menor nota.

# Como executar

1. Instale o Node.js (se ainda não tiver): https://nodejs.org/
2. No terminal, dentro da pasta do projeto, execute:
```bash
node notas-atletas.js
